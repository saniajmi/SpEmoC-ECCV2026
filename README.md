# SpEmoC-ECCV2026

## Dataset Access

The full SpEmoC dataset is available for non-commercial academic research.

### Access Procedure

1. Download the SpEmoC Access Request Form.
2. Complete and sign the form.
3. Submit the signed form to the corresponding authors using an official institutional email address.
4. Requests will be reviewed for compliance with the dataset license agreement.
5. Upon approval, access instructions and dataset download links will be provided.

### Why Controlled Access?

SpEmoC contains clips derived from publicly available movies and television content. To ensure responsible use and compliance with copyright and licensing considerations, direct public redistribution of the complete dataset is restricted.

The benchmark code, dataset documentation, metadata descriptions, and train/validation/test splits remain publicly accessible through this repository.
